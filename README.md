# Songpol's Data Analyst Portfolio

This repository hosts a simple, dark-themed portfolio website for Songpol, a Data Analyst. The site is designed to showcase analytical projects, dashboards, and key insights.

## Features

-   **Modern Dark Theme:** A sleek, eye-catching design.
-   **JetBrains Mono Typography:** Ensures code and data are readable and aesthetically pleasing.
-   **Hero Section:** Introduces Songpol as a Data Analyst.
-   **Projects Section:** Dedicated area to highlight 3 data analysis projects with detailed case studies.
-   **Contact Section:** Easy access to contact information (Email, LinkedIn, GitHub).

## Project Structure

-   `index.html`: The main landing page, integrating the Hero, Projects, and Contact sections.
-   `projects/`: Contains individual HTML files for detailed case studies of each project (e.g., `project1.html`).
-   `assets/`:
    -   `css/Style.css`: Stylesheets for the dark theme and typography.
    -   `js/theme.js`: (Placeholder for) JavaScript for interactive elements and data fetching.
    -   `images/`: (Planned for) Project screenshots and a profile photo.
    -   `data/`: (Planned for) JSON datasets or other relevant data files.
-   `plan/our_plan.md`: The original planning document for this portfolio.

## Setup and Development

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/songpolkhongyai-ctrl/Songpol-simple-portfolio.git
    cd Songpol-simple-portfolio
    ```
2.  **Open `index.html`:**
    Simply open the `index.html` file in your web browser to view the portfolio locally.

## Deployment

This project is intended to be deployed via GitHub Pages.

1.  Push your local changes to the `master` branch of your GitHub repository.
2.  Go to your repository settings on GitHub.
3.  Navigate to the "Pages" section.
4.  Select the `master` branch as your GitHub Pages source and choose the `/ (root)` folder.
5.  Save your changes. Your site should be live at `https://your-github-username.github.io/your-repository-name/`.
