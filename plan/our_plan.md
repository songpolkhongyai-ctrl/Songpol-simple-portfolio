# Portfolio Website Planning: Data Analysis Showcase

## Objective
Create a professional data analysis portfolio hosted on GitHub Pages to showcase analytical projects, dashboards, and insights.

## Specifications
- **Theme:** Modern Dark theme.
- **Typography:** JetBrains Mono font.
- **Hero Section:** Display "Songpol" (Data Analyst).
- **Projects:** Feature 3 data analysis projects.
- **Contact:** Include a dedicated section for contact details.

## Project Structure
- `index.html`: Main landing page (Hero, Projects, Contact).
- `projects/`: Folder for individual case studies.
- `assets/`:
    - `css/`: Stylesheets (`style.css` with dark theme and JetBrains Mono).
    - `js/`: Scripts for interactive elements/data fetching.
    - `data/`: JSON datasets.
    - `images/`: Project screenshots and profile photo.

## Implementation Steps
1. **Repository Setup:** Create a GitHub repository and initialize local files.
2. **Drafting:** Develop `index.html` with the specified Hero section, Projects section (3 slots), and Contact section.
3. **Styling:** Implement the Dark theme and apply JetBrains Mono font via CSS.
4. **Project Content:** Create the 3 project pages/sections with case study templates.
5. **Deployment:** Push to GitHub and enable GitHub Pages.

## Verification
- Confirm dark theme and font rendering.
- Verify all 3 projects are visible and interactive.
- Check contact links.
- Confirm site is live via GitHub Pages URL.
